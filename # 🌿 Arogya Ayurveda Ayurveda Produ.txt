# 🌿 Arogya Ayurveda: Ayurveda Products Store Website

**AWS S3 Hosting | Tailwind CSS | Responsive E-Commerce UI**

A modern and responsive Ayurveda Products Store designed to promote natural wellness through authentic Ayurvedic products. This project demonstrates a Serverless Static Website architecture hosted on Amazon S3, providing high availability, low maintenance, and cost-effective deployment.

---

## 🛠️ Technical Architecture

This project leverages AWS Cloud Infrastructure for scalable and reliable website hosting.

**Storage & Hosting:** Amazon S3 (Simple Storage Service)

**Region:** ap-south-1 (Mumbai)

**Frontend Framework:** Tailwind CSS (via CDN)

**Deployment Workflow:** Manual S3 Upload & Static Website Hosting

## ☁️ AWS Deployment Steps

To deploy this website on AWS:

### 1. S3 Bucket Creation

Created an S3 bucket with a unique name for static website hosting.

### 2. Static Website Hosting

Enabled Static Website Hosting and configured:

* Index Document: `index.html`
* Error Document: `index.html`

### 3. Public Access Configuration

* Disabled Block Public Access settings where required.
* Configured Bucket Policy for public read access.

### 4. Website Deployment

Uploaded the project files to the bucket root directory and verified website accessibility through the S3 Website Endpoint.
